CandyBloom v5 CMS

Dodano:
- panel admina /admin oparty o Decap CMS,
- edycję produktów w data/products.json,
- edycję ceny, opisu, kategorii, widoczności, zdjęcia, imagePosition i imageZoom,
- edycję zapachów, kategorii i opcji koloru w data/settings.json,
- upload zdjęć do assets/images.

Panel zapisuje zmiany dopiero po włączeniu w Netlify:
1. Identity
2. Invite only
3. Git Gateway
4. zaproszeniu Twojego maila jako użytkownika.

Po zapisaniu Decap CMS tworzy commit na GitHubie, a Netlify sam aktualizuje stronę.
