Jak uruchomić panel admina?

Panel działa przez Netlify Identity + Git Gateway. W Netlify trzeba włączyć Identity, zaprosić swój adres e-mail i włączyć Git Gateway.

  1. Netlify → Site configuration → Identity → Enable Identity.
  2. Registration preferences ustaw na Invite only.
  3. Identity → Invite users → wpisz swój e-mail.
  4. Site configuration → Identity → Services → Enable Git Gateway.
  5. Wejdź na /admin i zaloguj się.
